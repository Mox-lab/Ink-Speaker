# 人物档案

本文件是 RAG 设定库的人物档案示例。Agent 在写章节时可通过工具查询本文档,
确保人物言行与设定一致,避免人设崩塌。

## 1. 林晚(女主角)

- **年龄**:24 岁
- **身份**:云陵城码头脚夫,实为听潮阁外围线人
- **背景**:孤儿,自幼在码头长大,被前任阁主收为外围
- **性格**:外冷内热,记仇但讲原则;不善言辞,但观察力极强
- **外貌**:个子不高,常年穿灰色短打,左手腕有一道旧疤
- **武器**:短刀"霜序",前任阁主所赠,刀身刻"听潮"二字
- **禁忌**:不许任何人碰她的刀

## 2. 苏砚(男主角)

- **年龄**:27 岁
- **身份**:江湖游医,实为大燕皇室遗孤
- **背景**:幼年被忠臣带出宫,隐姓埋名学医,目的是查清父皇死因
- **性格**:温和寡言,凡事退一步;但触及底线时手段凌厉
- **外貌**:青衫长袍,随身带一只药箱,左手食指有针茧
- **武功**:不显,但暗藏内力,品阶疑似先天
- **秘密**:身上有皇室胎记,位于左肩胛

## 3. 赵九(反派)

- **年龄**:40 岁
- **身份**:锦衣卫百户,专司搜捕前朝余孽
- **背景**:出身寒微,凭军功晋升;亡妻死于前朝余孽之手,故对遗孤恨之入骨
- **性格**:多疑狠辣,但对亡妻一往情深;每年亡妻忌日必焚香独坐
- **外貌**:脸上一道刀疤,从左眉斜到右颊
- **武功**:宗师品阶,刀法"九转连环"
- **弱点**:亡妻留下的玉佩从不离身,可被利用

## 4. 关系网

- 林晚 ↔ 苏砚:第 1 章救人后相识,逐渐互生情愫;林晚不知苏砚身份。
- 林晚 ↔ 听潮阁:外围线人,与现任阁主有约定,不得泄露阁中事。
- 苏砚 ↔ 赵九:赵九奉命追捕,二人多次擦肩;最终决战在青州。
- 赵九 ↔ 林晚:赵九怀疑林晚藏匿苏砚,但拿不到证据。
